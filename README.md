# Conditional State

`Conditional State`는 `if`, `elif`, `else`, `and`, `or`, `not` 같은 조건문을
트럼프 카드 분류 게임으로 익히는 학습형 웹 서비스입니다.

학생은 공장 컨베이어벨트 위를 지나가는 플레잉 카드를 보고, 화면에 적힌 조건에 따라
어느 수직 벨트로 빠져야 하는지 직접 선택합니다. 정답이면 점수가 올라가고, 힌트를 보면
점수가 깎이는 방식으로 자연스럽게 조건 판단을 반복 연습할 수 있습니다.

## 서비스 개요

이 서비스는 설명을 길게 읽는 대신, 화면 안의 움직임 자체가 학습이 되도록 설계했습니다.

- 중앙 컨베이어벨트를 따라 카드가 이동합니다.
- 각 분기점에는 `if`, `elif`, `else`와 조건 토큰이 표지판처럼 배치됩니다.
- 사용자는 카드가 들어가야 할 수직 컨베이어벨트를 클릭합니다.
- 선택이 끝나면 카드가 실제 벨트를 따라 분기되며 결과를 보여 줍니다.
- 처음 진입 시에는 게임 화면 안에서 직접 벨트를 가리키는 튜토리얼이 표시됩니다.

## 학습 목표

이 프로젝트는 아래 개념을 시각적으로 익히는 데 초점을 맞춥니다.

- `if / else` 기본 분기
- `and` 와 `or`의 차이
- `not`으로 조건 뒤집기
- `if → elif → else`의 순차 실행
- 앞 조건이 먼저 참이면 뒤 조건은 보지 않는 실행 흐름

현재 문제는 쉬운 난이도부터 어려운 난이도 순으로 고정 배치되어 있으며,
총 10개의 미션으로 구성되어 있습니다.

## 플레이 방식

1. 시작 화면에서 게임을 시작합니다.
2. 현재 카드와 분기 조건을 읽습니다.
3. 정답이라고 생각하는 수직 컨베이어벨트를 클릭합니다.
4. 카드가 해당 벨트로 이동하는 애니메이션과 함께 결과를 확인합니다.
5. 정답이면 `10점`을 얻고, 힌트를 한 번 열 때마다 `2점`이 차감됩니다.
6. 모든 미션을 끝내면 마지막 화면에서 최종 점수를 확인할 수 있습니다.

## 현재 구성된 경험

- 플레잉 카드 형태의 카드 UI
- 흰 배경 기반의 컨베이어 작업장 화면
- 위/아래로 교차 배치된 수직 분기 벨트
- 벨트 클릭형 상호작용
- 친절한 말투의 피드백과 인게임 튜토리얼
- 힌트 보기 기능과 점수 차감 규칙
- 엔딩 화면의 최종 점수 표시

## 이 프로젝트가 지향하는 방식

이 앱은 조건문을 "설명"하는 데서 멈추지 않고, 학생이 직접 판단하고 결과를 보는
인터랙션으로 바꾸는 것을 목표로 합니다.  
즉, 코드를 먼저 보여 주는 학습이 아니라, 분기 흐름을 몸으로 이해하게 만드는
게임형 조건문 학습 도구입니다.
